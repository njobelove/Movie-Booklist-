# 🎬 Movie Booklist App

A simple web application to manage your personal movie watchlist. Keep track of movies you've watched, want to watch, and rate them.

## ✨ Features

- Add movies to your booklist with title, year, genre, and rating.
- Mark movies as "watched" or "to watch".
- Edit or delete movie entries.
- Search and filter movies by title or genre.
- Responsive design for mobile and desktop.
